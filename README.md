# Shadow's Sirocco Mod

A mod that fixes several issues/bugs in the P2P version of Sirocco:

- Disables the F1 debug hotkey (this used to freeze your game if accidentally pressed mid-game)
- Scoreboard now works for all clients joining your lobbies (this includes post-game analysis/statistics)
- Fixes the in-game chat for all clients joining your lobbies

## SiroccoSkins.dll (optional)

This is an optional skin mod that applies random Sampan skins to all players. These are visible to all players that join your lobbies and makes it easier to distinguish between players in the early game.
